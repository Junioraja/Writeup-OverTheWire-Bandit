## Deskripsi singkat

Di level ini password untuk level berikutnya disimpan dalam file data.txt  
Isi file tersebut sudah diencode menggunakan base64 sehingga tidak bisa langsung dibaca  
Saya perlu melakukan proses decode untuk mendapatkan password aslinya  

---

## 1 Login ke server

Saya login menggunakan password dari level sebelumnya  

```bash
ssh bandit10@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat isi direktori  

```bash
ls
```

Terlihat file bernama data.txt  

---

## 2 Melihat isi file

Saya mencoba melihat isi file  

```bash
cat data.txt
```

Output menampilkan teks acak yang merupakan hasil encoding base64  

---

## 3 Melakukan decode base64

Karena file tersebut menggunakan base64 saya melakukan decode dengan perintah  

```bash
base64 -d data.txt
```

Setelah menjalankan perintah tersebut muncul string yang bisa dibaca  

String tersebut adalah password untuk bandit11  

---

## Kesimpulan

Di level ini saya belajar mengenali format encoding base64  
Saya juga belajar menggunakan perintah base64 dengan opsi -d untuk melakukan decoding  
Level ini melatih pemahaman tentang encoding dan decoding data di linux
