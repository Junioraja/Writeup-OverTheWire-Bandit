Platform: OverTheWire  
Wargame: Bandit  

## Tujuan

Login ke server dan menemukan password untuk level 1.

---

## Deskripsi Singkat

Level 0 merupakan level pengenalan. Pada level ini, password untuk level berikutnya disimpan dalam sebuah file di home directory user.

---

## 1. Login ke Server

Buka terminal lalu jalankan perintah berikut:

```bash
ssh bandit0@bandit.labs.overthewire.org -p 2220
```

Jika muncul pesan:

```
Are you sure you want to continue connecting?
```

Ketik:

```
yes
```

Password untuk level 0 adalah:

```
bandit0
```

Jika login berhasil akan masuk ke shell sebagai user `bandit0`.

---

## 2. Melihat Isi Direktori

Setelah login, jalankan perintah berikut untuk melihat isi direktori:

```bash
ls
```

Output:

```
readme
```

Terdapat satu file bernama `readme`.

---

## 3. Membaca Isi File

Gunakan perintah berikut untuk membaca isi file:

```bash
cat readme
```

Perintah tersebut akan menampilkan sebuah string panjang.  
String tersebut adalah password untuk user `bandit1`.

---

## 4. Logout dan Login ke Level Berikutnya

Keluar dari sesi SSH:

```bash
exit
```

Kemudian login kembali menggunakan user `bandit1` dan password yang telah ditemukan:

```bash
ssh bandit1@bandit.labs.overthewire.org -p 2220
```

---

## Konsep yang Dipelajari

- Menggunakan SSH untuk remote login
- Navigasi dasar sistem Linux
- Menggunakan perintah `ls`
- Membaca file dengan perintah `cat`
- Memahami struktur home directory pada sistem Linux
