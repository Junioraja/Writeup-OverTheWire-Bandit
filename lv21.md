## Deskripsi singkat

Di level ini terdapat sebuah cron job yang berjalan secara berkala  
Cron job tersebut mengeksekusi sebuah script yang dapat saya baca  
Password untuk level berikutnya disimpan di lokasi yang ditentukan oleh script tersebut  
Artinya saya perlu menganalisis isi script cron yang berjalan  

---

## 1 Login ke server

Saya login sebagai bandit21 menggunakan password dari level sebelumnya  

```bash
ssh bandit21@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya mencari informasi tentang cron job  

---

## 2 Mengecek direktori cron

Saya melihat isi direktori cron  

```bash
ls /etc/cron.d/
```

Di sana terdapat file bernama cronjob_bandit22  

Saya membaca isi file tersebut  

```bash
cat /etc/cron.d/cronjob_bandit22
```

Output menunjukkan bahwa sistem menjalankan script tertentu setiap menit  

---

## 3 Membaca isi script

Saya melihat path script yang dijalankan lalu membacanya  

```bash
cat /usr/bin/cronjob_bandit22.sh
```

Dari isi script terlihat bahwa password untuk bandit22 disalin ke file tertentu di direktori /tmp  

---

## 4 Mengambil password

Saya membuka file yang disebutkan dalam script tersebut  

```bash
cat /tmp/nama_file_yang_disebutkan
```

Isi file tersebut adalah string yang merupakan password untuk bandit22  

---

## Kesimpulan

Di level ini saya belajar memahami cara kerja cron job di linux  
Saya menganalisis script yang dijalankan secara otomatis untuk menemukan lokasi password  
Level ini melatih kemampuan membaca script dan memahami alur eksekusi sistem
