## Deskripsi singkat

Di level ini setiap kali saya login menggunakan ssh koneksi langsung tertutup  
Hal ini terjadi karena ada konfigurasi tertentu yang membuat shell langsung keluar  
Untuk mengatasinya saya perlu menjalankan perintah tertentu saat login tanpa masuk ke shell interaktif  

---

## 1. Login dengan menjalankan perintah langsung

Karena jika login biasa langsung logout maka saya menjalankan perintah saat koneksi dibuat  

```bash
ssh bandit18@bandit.labs.overthewire.org -p 2220 cat readme
```

Dengan cara ini saya tidak masuk ke shell interaktif tetapi langsung mengeksekusi perintah cat readme  

---

## 2. Mendapatkan password

Perintah tersebut langsung menampilkan isi file readme  

String yang muncul adalah password untuk bandit19  

---

## Kesimpulan

Di level ini saya belajar bahwa ssh bisa digunakan untuk langsung menjalankan perintah tanpa membuka shell  
Teknik ini berguna ketika akses interaktif dibatasi atau langsung ditutup  
Level ini melatih pemahaman tentang fleksibilitas penggunaan ssh di linux
