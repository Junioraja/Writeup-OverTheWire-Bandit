## Tujuan

## Deskripsi singkat

Di level ini password untuk level berikutnya berada di dalam direktori inhere  
Di dalam folder tersebut terdapat banyak file dan hanya satu file yang berisi teks yang bisa dibaca manusia  
File lainnya berisi data biner  

---

## 1 Login ke server

Saya login menggunakan password dari level sebelumnya  

```bash
ssh bandit4@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya berada di home directory  

---

## 2 Masuk ke direktori inhere

Saya melihat isi direktori  

```bash
ls
```

Output menunjukkan folder bernama inhere  

Saya masuk ke folder tersebut  

```bash
cd inhere
```

Kemudian saya cek isinya  

```bash
ls
```

Terlihat banyak file dengan nama seperti -file00 -file01 dan seterusnya  

---

## 3 Mengecek tipe file

Karena hanya satu file yang berisi teks biasa saya menggunakan perintah file untuk mengecek tipe masing masing file  

```bash
file ./*
```

Output akan menampilkan tipe dari setiap file  

Dari hasil tersebut terlihat satu file yang bertipe ASCII text  

---

## 4 Membaca file yang benar

Setelah mengetahui file mana yang bertipe teks saya membaca file tersebut dengan  

```bash
cat ./nama_file
```

Kemudian muncul string panjang yang merupakan password untuk bandit5  

---

## Kesimpulan

Di level ini saya belajar menggunakan perintah file untuk mengidentifikasi tipe data suatu file  
Saya juga belajar bagaimana menangani banyak file sekaligus menggunakan wildcard seperti .*  
Level ini melatih pemahaman tentang perbedaan file teks dan file biner di linux
