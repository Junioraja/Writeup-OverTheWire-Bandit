## Tujuan

Mendapatkan password untuk level 18

## Deskripsi singkat

Di level ini terdapat dua file yaitu passwords.old dan passwords.new  
Password untuk level berikutnya adalah baris yang berbeda antara kedua file tersebut  
Artinya saya perlu membandingkan isi kedua file  

---

## 1. Login ke server

Saya login sebagai bandit17 menggunakan private key yang didapat dari level sebelumnya  

```bash
ssh -i key.txt bandit17@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat isi direktori  

```bash
ls
```

Terlihat dua file yaitu passwords.old dan passwords.new  

---

## 2. Membandingkan kedua file

Untuk melihat perbedaan isi kedua file saya menggunakan perintah diff  

```bash
diff passwords.old passwords.new
```

Output menampilkan perbedaan antara kedua file  

Baris yang hanya muncul di passwords.new dan tidak ada di passwords.old adalah password untuk bandit18  

---

## 3. Mengambil password

Dari hasil diff saya menyalin string yang berbeda tersebut  

String itu merupakan password untuk level 18  

---

## Kesimpulan

Di level ini saya belajar menggunakan perintah diff untuk membandingkan dua file  
Saya memahami bagaimana mencari perbedaan antar file secara efisien tanpa membaca satu per satu  
Level ini melatih kemampuan analisis perubahan data di linux
