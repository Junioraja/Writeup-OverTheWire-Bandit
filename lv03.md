## Tujuan

Mencari password untuk level 4  

---

## Deskripsi singkat

Di level ini password untuk level berikutnya disimpan di dalam sebuah direktori  
File tersebut tersembunyi sehingga tidak langsung terlihat saat menggunakan perintah ls biasa  

---

## 1 Login ke server

Saya login menggunakan password yang didapat dari level sebelumnya  

```bash
ssh bandit3@bandit.labs.overthewire.org -p 2220
```

Setelah memasukkan password level 3 saya berhasil masuk  

---

## 2 Mengecek isi direktori

Saya melihat isi direktori dengan  

```bash
ls
```

Output yang muncul  

```
inhere
```

Terdapat satu folder bernama inhere  

---

## 3 Masuk ke direktori

Saya masuk ke folder tersebut  

```bash
cd inhere
```

Kemudian saya cek isinya  

```bash
ls
```

Tidak ada file yang terlihat  

---

## 4 Menampilkan file tersembunyi

Karena tidak ada file yang terlihat saya mencoba menampilkan file tersembunyi dengan  

```bash
ls -a
```

Output yang muncul  

```
.
..
.hidden
```

Terlihat ada file bernama .hidden  

---

## 5 Membaca isi file

Saya membaca file tersebut dengan  

```bash
cat .hidden
```

Setelah itu muncul string panjang yang merupakan password untuk bandit4  

---

## Kesimpulan

Di level ini saya belajar bahwa file yang diawali tanda titik dianggap sebagai file tersembunyi  
Untuk melihatnya perlu menggunakan opsi -a pada perintah ls  
Level ini melatih pemahaman tentang hidden file di linux
