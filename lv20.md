## Deskripsi singkat

Di level ini terdapat file executable bernama suconnect  
Program tersebut akan terhubung ke port tertentu di localhost lalu membaca password yang dikirim  
Jika password yang dikirim benar maka program akan mengembalikan password untuk level berikutnya  
Artinya saya perlu membuat service yang mendengarkan pada port tertentu lalu mengirimkan password level saat ini  

---

## 1 Login ke server

Saya login sebagai bandit20 menggunakan password dari level sebelumnya  

```bash
ssh bandit20@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat isi direktori  

```bash
ls -la
```

Terlihat file executable bernama suconnect  

---

## 2 Menjalankan listener

Karena program akan terhubung ke port yang saya tentukan maka saya perlu membuka listener terlebih dahulu  
Saya menggunakan netcat dan memilih port bebas misalnya 4444  

```bash
nc -lvp 4444
```

Terminal ini saya biarkan terbuka  

---

## 3 Menjalankan program suconnect

Di terminal lain saya menjalankan  

```bash
./suconnect 4444
```

Program tersebut akan mencoba terhubung ke listener yang saya buat  

---

## 4 Mengirim password

Saat koneksi terbentuk saya mengirim password level 20 melalui terminal listener lalu menekan enter  

Jika password benar maka suconnect akan merespons dengan string baru  

String tersebut adalah password untuk bandit21  

---

## Kesimpulan

Di level ini saya belajar tentang konsep client dan server sederhana  
Saya memahami bagaimana membuat listener menggunakan netcat dan menerima koneksi dari program lain  
Level ini melatih pemahaman tentang komunikasi jaringan lokal dan interaksi antar proses
