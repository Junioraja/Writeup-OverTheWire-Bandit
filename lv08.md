## Deskripsi singkat

Di level ini password untuk level berikutnya disimpan dalam file data.txt  
Password tersebut adalah satu satunya baris yang muncul satu kali saja  
Artinya semua baris lain muncul lebih dari satu kali  

---

## 1. Login ke server

Saya login menggunakan password dari level sebelumnya  

```bash
ssh bandit8@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat isi direktori  

```bash
ls
```

Terlihat file bernama data.txt  

---

## 2. Mencari baris yang unik

Karena password adalah baris yang hanya muncul satu kali saya perlu memfilter baris yang unik  
Saya menggunakan kombinasi sort dan uniq  

```bash
sort data.txt | uniq -u
```

Penjelasan singkat  
sort digunakan untuk mengurutkan isi file agar baris yang sama berdekatan  
uniq -u digunakan untuk menampilkan hanya baris yang tidak memiliki duplikat  

Setelah menjalankan perintah tersebut muncul satu baris string panjang  

String tersebut adalah password untuk bandit9
