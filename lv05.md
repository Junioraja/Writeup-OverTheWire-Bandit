## Deskripsi singkat

Di level ini password berada di dalam direktori inhere  
File yang dicari memiliki kriteria tertentu yaitu ukurannya 1033 bytes tidak dapat dieksekusi dan tidak dapat dibaca oleh user lain  
Karena jumlah file cukup banyak maka perlu menggunakan perintah pencarian  

---

## 1 Login ke server

Saya login menggunakan password dari level sebelumnya  

```bash
ssh bandit5@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat isi direktori  

```bash
ls
```

Terdapat folder bernama inhere lalu saya masuk ke folder tersebut  

```bash
cd inhere
```

---

## 2 Mencari file sesuai kriteria

Karena file yang dicari memiliki ukuran spesifik yaitu 1033 bytes saya menggunakan perintah find  

```bash
find . -type f -size 1033c
```

Penjelasan singkat  
-type f untuk mencari file biasa  
-size 1033c untuk mencari file berukuran 1033 bytes  

Dari hasil tersebut muncul satu path file yang sesuai  

---

## 3 Membaca file yang ditemukan

Setelah mengetahui lokasi file saya membaca isinya dengan  

```bash
cat ./path_ke_file
```

Kemudian muncul string panjang yang merupakan password untuk bandit6  

---

## Kesimpulan

Di level ini saya belajar menggunakan perintah find untuk mencari file berdasarkan ukuran dan tipe  
Level ini melatih pemahaman tentang pencarian file secara spesifik di sistem linux terutama ketika jumlah file sangat banyak
