## Deskripsi singkat

Di level ini password untuk level berikutnya disimpan dalam file data.txt  
File tersebut berisi banyak karakter yang tidak terbaca karena merupakan data biner  
Password berada di salah satu string yang didahului oleh beberapa tanda sama dengan  

---

## 1. Login ke server

Saya login menggunakan password dari level sebelumnya  

```bash
ssh bandit9@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat isi direktori  

```bash
ls
```

Terlihat file bernama data.txt  

---

## 2. Mengekstrak teks yang bisa dibaca

Karena file berisi banyak karakter aneh saya tidak bisa langsung menggunakan cat  
Saya menggunakan perintah strings untuk mengambil bagian teks yang bisa dibaca manusia  

```bash
strings data.txt
```

Output menampilkan banyak baris teks  

---

## 3. Mencari baris yang mengandung tanda sama dengan

Dari deskripsi level diketahui password berada di string yang didahului beberapa tanda sama dengan  
Saya menggabungkan strings dengan grep untuk memfilter baris yang relevan  

```bash
strings data.txt | grep "==="
```

Setelah menjalankan perintah tersebut muncul beberapa baris  

Salah satu baris berisi string panjang yang merupakan password untuk bandit10  

---
