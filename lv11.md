## Tujuan

Mencari password untuk level 12

## Deskripsi singkat

Di level ini password untuk level berikutnya disimpan dalam file data.txt  
Isi file tersebut sudah dienkripsi menggunakan rot13  
Artinya setiap huruf digeser 13 posisi dalam alfabet  
Untuk mendapatkan password asli saya perlu mengembalikannya ke bentuk semula  

---

## 1 Login ke server

Saya login menggunakan password dari level sebelumnya  

```bash
ssh bandit11@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat isi direktori  

```bash
ls
```

Terlihat file bernama data.txt  

---

## 2 Melihat isi file

Saya membaca isi file terlebih dahulu  

```bash
cat data.txt
```

Output menampilkan string yang masih terenkripsi dengan rot13  

---

## 3 Melakukan decode rot13

Untuk mengembalikan teks ke bentuk asli saya menggunakan perintah tr  

```bash
cat data.txt | tr 'A-Za-z' 'N-ZA-Mn-za-m'
```

Perintah tersebut menggeser kembali huruf sebanyak 13 posisi  

Setelah dijalankan muncul string yang merupakan password untuk bandit12

Di level ini saya belajar tentang teknik enkripsi sederhana bernama rot13  
Saya juga belajar menggunakan perintah tr untuk melakukan translasi karakter  
Level ini melatih pemahaman tentang manipulasi teks di linux
