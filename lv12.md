## Tujuan

Mencari password untuk level 13

## Deskripsi singkat

Di level ini password untuk level berikutnya disimpan dalam file data.txt  
File tersebut merupakan hasil kompresi berulang kali dalam beberapa format berbeda  
Untuk mendapatkan password saya perlu mengekstrak file tersebut secara bertahap sampai menemukan file teks aslinya  

---

## 1 Login ke server

Saya login menggunakan password dari level sebelumnya  

```bash
ssh bandit12@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat file data.txt  

```bash
ls
```

---

## 2 Membuat direktori kerja

Agar tidak berantakan saya membuat folder sementara di direktori /tmp  

```bash
mkdir /tmp/bandit12
cd /tmp/bandit12
```

Lalu saya menyalin file data.txt ke folder tersebut  

```bash
cp ~/data.txt .
```

---

## 3 Mengubah data menjadi file biner

Saat dicek dengan file terlihat bahwa data.txt merupakan hexdump  
Saya perlu mengubahnya kembali menjadi bentuk biner  

```bash
xxd -r data.txt > data.bin
```

Setelah itu saya mengecek tipe file  

```bash
file data.bin
```

---

## 4 Mengekstrak file secara bertahap

Dari hasil file terlihat jenis kompresinya  
Saya mengekstraknya sesuai tipe yang muncul  

Jika gzip  

```bash
mv data.bin data.gz
gunzip data.gz
```

Jika bzip2  

```bash
mv data.bin data.bz2
bunzip2 data.bz2
```

Jika tar  

```bash
tar -xf data.bin
```

Setiap selesai ekstraksi saya kembali menjalankan  

```bash
file nama_file
```

Lalu mengulangi proses sampai akhirnya mendapatkan file teks biasa  

---

## 5 Membaca password

Setelah file terakhir berhasil diekstrak dan bertipe ASCII text saya membacanya dengan  

```bash
cat nama_file
```

Output menampilkan string yang merupakan password untuk bandit13  

---

## Kesimpulan

Di level ini saya belajar tentang berbagai jenis kompresi seperti gzip bzip2 dan tar  
Saya juga belajar menggunakan xxd untuk mengembalikan hexdump menjadi file asli  
Level ini melatih ketelitian dan pemahaman tentang proses ekstraksi file bertahap di linux
