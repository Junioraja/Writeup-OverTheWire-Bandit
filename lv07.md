## Tujuan

## Deskripsi singkat

Di level ini password untuk level berikutnya disimpan dalam file bernama data.txt  
Password berada pada baris yang mengandung kata millionth  
Karena isi file cukup banyak maka perlu menggunakan pencarian teks  

---

## 1. Login ke server

Saya login menggunakan password dari level sebelumnya  

```bash
ssh bandit7@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya melihat isi direktori  

```bash
ls
```

Terlihat file bernama data.txt  

---

## 2. Membaca isi file

Karena file kemungkinan besar berisi banyak baris saya tidak membaca seluruhnya dengan cat  
Saya langsung mencari baris yang mengandung kata millionth menggunakan perintah grep  

```bash
grep "millionth" data.txt
```

Perintah tersebut menampilkan satu baris yang berisi kata millionth diikuti string panjang  

String tersebut adalah password untuk bandit8  

---

## Kesimpulan

Di level ini saya belajar menggunakan grep untuk mencari teks tertentu di dalam file  
Level ini melatih kemampuan mencari informasi spesifik di antara banyak data
