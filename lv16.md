## Deskripsi singkat

Di level ini saya diminta untuk mencari port yang menerima koneksi ssl di rentang port 31000 sampai 32000  
Salah satu port akan mengembalikan private key jika saya mengirim password level saat ini  
Karena jumlah port cukup banyak saya perlu melakukan scanning terlebih dahulu  

---

## 1. Login ke server

Saya login sebagai bandit16 menggunakan password dari level sebelumnya  

```bash
ssh bandit16@bandit.labs.overthewire.org -p 2220
```

Setelah berhasil masuk saya siap melakukan scanning  

---

## 2. Melakukan scanning port

Untuk mencari port mana yang aktif dan menggunakan ssl saya menggunakan nmap  

```bash
nmap -p 31000-32000 localhost
```

Dari hasil scanning terlihat beberapa port terbuka  

Untuk memastikan mana yang menggunakan ssl saya menambahkan opsi ssl detection  

```bash
nmap -p 31000-32000 -sV localhost
```

Dari hasil tersebut saya menemukan satu port yang menggunakan ssl  

---

## 3. Mengirim password ke port yang benar

Setelah mengetahui port yang tepat saya terhubung menggunakan openssl  

```bash
openssl s_client -connect localhost:port_yang_ditemukan
```

Kemudian saya mengirim password level 16  

Jika benar server akan mengembalikan private key  

---

## 4. Menggunakan private key

Saya menyimpan private key tersebut ke dalam file disini contohnya key.txt lalu mengatur permission agar aman  

```bash
chmod 600 key.txt
```

Kemudian saya login sebagai bandit17 menggunakan key tersebut  

```bash
ssh -i key.txt bandit17@localhost -p 2220
```

Setelah berhasil login saya mendapatkan akses ke level berikutnya  

---

## Kesimpulan

Di level ini saya belajar melakukan scanning port menggunakan nmap  
Saya juga belajar mengidentifikasi service ssl dan menggunakan openssl untuk berinteraksi  
Level ini melatih pemahaman tentang enumerasi service dan penggunaan private key untuk autentikasi ssh
