## Deskripsi singkat

Di level ini password untuk level berikutnya tidak disimpan dalam file  
Saya diminta untuk mengirim password level saat ini ke sebuah service yang berjalan di localhost pada port tertentu  
Jika password benar maka service akan mengembalikan password untuk level berikutnya  

---

## 1. Login ke server

Saya login menggunakan metode yang sama seperti akhir level sebelumnya  

```bash
ssh bandit14@bandit.labs.overthewire.org -p 2220
```

Atau jika masih berada di server sebelumnya saya langsung menggunakan sesi yang sudah login sebagai bandit14  

---

## 2. Mengirim password ke service

Dari deskripsi level diketahui bahwa saya harus mengirim password level 14 ke localhost pada port 30000  

Saya menggunakan netcat untuk terhubung ke port tersebut  

```bash
nc localhost 30000
```

Setelah koneksi terbuka saya menempelkan password level 14 lalu menekan enter  

---

## 3. Mendapatkan password baru

Jika password yang saya kirim benar maka service akan merespons dengan string baru  

String tersebut adalah password untuk bandit15  

---

## Kesimpulan

Di level ini saya belajar menggunakan netcat untuk berinteraksi dengan service melalui port tertentu  
Saya memahami konsep komunikasi client dan server sederhana melalui koneksi tcp  
Level ini melatih pemahaman tentang dasar networking di linux
